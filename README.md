For this assignment I chose to integrate two external API's to my fictional creative studio site based in Los Angeles, California.
I integrated a weather API from api.weather.gov that displays the fictional company's location, temperature, and weather description. 
The second API is from newsapi.org, which displays news article headlines, filtered to display art and design related headlines to fit with the website's theme. 
The weather API is keyless so it appears immediately upon deployment. For the news API, I integreated the key and its variable name in the site configuration on Netlify. Uses do not need to do anything to make it run.
